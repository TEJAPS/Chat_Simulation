##Installation

### Requirements
* PHP >= 5.6.4
* OpenSSL PHP Extension
* PDO PHP Extension
* Mbstring PHP Extension
* Tokenizer PHP Extension
* XML PHP Extension

### Terminal

```
$ cp .env.example .env
```

```
$ php artisan key:generate
```

```
$ composer install
```
